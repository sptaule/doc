# Introduction

:::tip Bienvenue ! :tada:

Je vous souhaite la bienvenue sur la documentation de Diveblob.<br/>
Si vous voulez retrouver le détail de toutes les fonctionnalités vous êtes au bon endroit.
:::

## Qu'est-ce que c'est ?

**Diveblob** est une solution de gestion pour les clubs de plongée.

Elle permet principalement de gérer des membres, des évènements, générer des documents.<br/>
Aussi, il y a la possibilité de proposer des achats en ligne permettant de créditer les plongeurs.


Le tout pensé pour une automatisation et une simplification maximales !

## Pour qui s'adresse-t-elle ?

Pour les dirigeants de clubs de plongée !

Si vous êtes `pressé`, `fatigué` et si vous trouvez que c'est une activité `chronophage`, `pénible`, vous devriez trouver satisfaction avec cette solution :blush:

Vous n'avez pas besoin de connaissances particulières.

## Démarrer en tant qu'administrateur

En premier lieu, vous devez vous connecter.<br/>
Pour se faire, rendez-vous sur l'adresse :
```shell
https://www.votresite.com/admin
```

Puis avec vos identifiants, après une connexion réussie, vous arrivez sur le tableau de bord.<br/>
C'est ici que vous pouvez avoir une vue globale sur le club et accéder aux différentes rubriques.
