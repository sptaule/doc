## Qu'est-ce que c'est ?

**Scuba** est une solution de gestion pour les clubs de plongée.

Elle permet principalement de gérer des membres, des évènements, générer des documents.<br/>
Aussi, il y a la possibilité de proposer des achats en ligne permettant de créditer les plongeurs.<br/>
Ces crédits permettent de s'inscrire aux différents évènements publiés sur le site.


Le tout pensé pour une automatisation et une simplification maximales !

## Pour qui s'adresse-t-elle ?

Pour les dirigeants de clubs de plongée !

Si vous êtes **pressé** ou si vous trouvez que c'est une activité **chronophage** de gérer un site web, vous devriez être satisfait avec cette solution :blush:

Vous avez seulement besoin de <u>connaissances de base</u> en informatique.

## Démarrer en tant qu'administrateur

En premier lieu, vous devez vous connecter.<br/>
Pour se faire, rendez-vous sur l'adresse :
```shell
https://www.votresite.com/admin
```

Puis avec vos identifiants, après une connexion réussie, vous arrivez sur le tableau de bord.<br/>
C'est ici que vous pouvez avoir une vue globale sur le club et accéder aux différentes rubriques.
